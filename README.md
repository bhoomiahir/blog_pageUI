# Mind Diary Blog Webpage 
Mind Diary is a general blog webpage created using HTML and CSS. It presents articles related to technology, creativity and learning in a clean and minimal layout. 
## Features 
- Responsive blog layout
- Header with navigation bar
- Main blog content section
- Sidebar with categories and recent posts
- Styled footer with branding
- Clean UI using CSS Grid and Flexbox
## Technologies Used 
- HTML5
- CSS3
- CSS Grid
- Flexbox
## How to Run 
1. Download or clone the repository
2. Open the project folder
3. Open index.html in a web browser
## Purpose 
This project was created as part of a web development learning assignment to practice layout design and responsive structure.
